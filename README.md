# Introduction about this Project
这个项目是基于亚马逊以及其他美国主流电商平台上的顾客评价，对Oral-B电动牙刷的产品表现进行分析，从而帮助企业更好的改进产品，高效竞争。
主要使用到的技术包括：
1. 网络爬虫 Web Crawling
2. 自然语言处理 NLP
3. 机器学习&逻辑回归 Machine Learning & Logistic Regression
4. 数据可视化 Data Visualization
